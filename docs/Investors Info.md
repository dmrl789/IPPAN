### HashTimer™ Mechanism (Patent Pending)

**Problem:** Distributed systems have no shared notion of "now"
**Solution:** Cryptographic median consensus on temporal order

**Innovation:**
1. Nodes broadcast local timestamps with cryptographic signatures
2. Network computes weighted median (reputation-adjusted)
3. Resulting HashTimer becomes immutable ordering reference
4. All events deterministically sequenced by HT value

**Competitive Moat:**
- Patent protection (USPTO filing)
- 3+ years R&D head start
- Impossible to replicate without license
- Network effects compound advantage

### Roundchain Consensus

Traditional blockchains are **sequential** (one block at a time).
IPPAN is **parallel** (thousands of micro-blocks simultaneously).

**Mechanism:**
- Validators propose blocks every 10-50ms
- Each block references multiple parents (DAG structure)
- HashTimer orders all blocks deterministically every 100-250ms
- Aggregated into RoundRoot with zk-STARK proof

**Advantage:** Linear scaling with validator count (not bottlenecked by sequential ordering)
