# 🌐 **IPPAN Top-Level Domain Registry**

## Overview

The IPPAN TLD Registry contains a comprehensive list of available top-level domains for the IPPAN network. This registry excludes existing DNS TLDs and ISO country codes to avoid conflicts while providing a rich ecosystem of domain options.

## CLI Usage

### List Available TLDs
```bash
# List all available TLDs
ippan list-tlds

# List TLDs by category
ippan list-tlds --category tech

# List TLDs by premium multiplier
ippan list-tlds --multiplier 10

# Show detailed information
ippan list-tlds --detailed
```

### Register Domains
```bash
# Register standard domain
ippan domain register example.ipn --years 1

# Register premium domain
ippan domain register example.ai --years 1

# Register tech domain
ippan domain register example.dlt --years 1

# Register IoT domain
ippan domain register example.iot --years 1
```

## Registry Management

### Adding New TLDs
New TLDs can be added to the registry through governance proposals. The process includes:

1. **Proposal Submission** - Community member submits TLD proposal
2. **Technical Review** - Validation of TLD format and conflicts
3. **Governance Vote** - Community votes on TLD addition
4. **Implementation** - TLD added to registry with appropriate multiplier

### TLD Categories
TLDs are categorized based on:
- **Length** - 1-2 letter TLDs are premium
- **Purpose** - Tech, finance, IoT, etc.
- **Market Value** - Demand and branding potential
- **Network Impact** - Contribution to ecosystem

### Premium Multiplier Rules
- **×10**: 1-2 letter TLDs, premium branding
- **×5**: Technology-focused domains
- **×3**: Finance and decentralized domains
- **×2**: IoT, storage, development domains
- **×1**: Standard domains

## Technical Implementation

### Registry Structure
```rust
pub struct TldRegistry {
    pub tlds: HashMap<String, TldEntry>,
    pub default_tld: String,
}

pub struct TldEntry {
    pub tld: String,
    pub premium_multiplier: u32,
    pub category: TldCategory,
    pub description: String,
    pub available: bool,
    pub min_years: u32,
    pub max_years: u32,
}
```

### Fee Calculation
```rust
pub fn domain_fee(year: u32, premium_mult: u32) -> u64 {
    let base: f64 = if year == 1 {
        0.20
    } else if year == 2 {
        0.02
    } else {
        let decayed = 0.01 - 0.001 * (year as f64 - 3.0);
        if decayed < 0.001 { 0.001 } else { decayed }
    };
    (base * premium_mult as f64 * 1_000_000.0).round() as u64
}
```
